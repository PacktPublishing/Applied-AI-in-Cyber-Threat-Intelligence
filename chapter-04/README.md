# Chapter 4 — Data Science Foundations

A hands-on orientation to the data science stack covering concurrency, statistics, machine learning, and simulation — applied through a series of progressive Python examples.

## What This Chapter Covers

| Section | Topic |
|---|---|
| 4.1 | Data Science Lifecycle |
| 4.2 | Configuring Your Environment (Jupyter/Colab + VS Code) |
| 4.3 | Concurrency and Data Manipulation (Threading, Multiprocessing, Pandas) |
| 4.4 | Statistics Foundation (Descriptive Stats, EDA, ydata-profiling) |
| 4.5 | Classical Machine Learning (Supervised and Unsupervised) |
| 4.6 | Machine Learning Metrics (Accuracy, Precision, Recall, F1, MAE) |
| 4.7 | Visualization with Matplotlib |
| 4.8 | Monte Carlo Simulations |

## Notebook

All code examples are in `Chapter_4_Work.ipynb`. The notebook is designed to be run top to bottom in Google Colab or JupyterLab.
